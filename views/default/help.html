Using SeqConfig

Create New Runs

Please download a template, fill up details and upload.

Edit Run

Click on button 'Show Lanes'. Add, edit and delete entries as needed.

Delete Run

Click on button 'Delete Run' to delete.

Create Readme

Clicking on button 'Create Readme' validates entries for a Run and uploads a Readme file to server with status 'Waiting for Approval' Once this run is approved by an Admin, it is copied to server to start preprocessing and a mail is sent to Admins. All entries in a table can be exported as csv format files All previous runs are saved in the database. For bugs, suggestions and comments, please mail pmakella@uchicago.edu
